**Simple Loop in Python**
```python
# Example 1: For Loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Output:
# apple
# banana
# cherry

# Example 2: While Loop
i = 0
while i < 5:
    print(i)
    i += 1

# Output:
# 0
# 1
# 2
# 3
# 4
```

**Simple Loop with Conditional Statement**
```python
# Example 1: For Loop with Conditional Statement
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    if fruit == "banana":
        print("I love", fruit)
    else:
        print(fruit)

# Output:
# apple
# I love banana
# cherry

# Example 2: While Loop with Conditional Statement
i = 0
while i < 5:
    if i == 3:
        print("Stop here!")
        break
    print(i)
    i += 1

# Output:
# 0
# 1
# 2
# Stop here!
```

**Simple Loop with Range Function**
```python
# Example: For Loop with Range Function
for i in range(5):
    print(i)

# Output:
# 0
# 1
# 2
# 3
# 4

# Example: While Loop with Range Function
i = 0
while i < 5:
    print(i)
    i += 1

# Output:
# 0
# 1
# 2
# 3
# 4
```

**Simple Loop with Enumerate Function**
```python
# Example: For Loop with Enumerate Function
fruits = ["apple", "banana", "cherry"]
for i, fruit in enumerate(fruits):
    print(f"Index: {i}, Fruit: {fruit}")

# Output:
# Index: 0, Fruit: apple
# Index: 1, Fruit: banana
# Index: 2, Fruit: cherry
```

**Simple Loop with Zip Function**
```python
# Example: For Loop with Zip Function
fruits = ["apple", "banana", "cherry"]
quantities = [5, 10, 15]
for fruit, quantity in zip(fruits, quantities):
    print(f"Fruit: {fruit}, Quantity: {quantity}")

# Output:
# Fruit: apple, Quantity: 5
# Fruit: banana, Quantity: 10
# Fruit: cherry, Quantity: 15
```
